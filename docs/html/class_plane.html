Detailed Description

The GeometricObject representing a Plane.

The plane is defined by a point a and a normal n. So any point p will lie inside the plane if the normal is perpendicular on the vector \(p - a\). So the equation of the plane is:

\[ (p - a) \cdot n = 0 \]
